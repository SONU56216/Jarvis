# JARVIS AI Assistant

JARVIS is a personal AI voice assistant built with Python.
It can perform voice commands, manage tasks, play music, control apps, and work as a smart desktop assistant.

## Features

* Voice recognition
* AI chat system
* Music player
* Task & notes manager
* Alarm and reminder system
* Mood and routine tracking
* Phone control with ADB
* Basic robotics support
* Code assistant features

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Tkinter
* Google Gemini AI
* OpenCV
* Pygame
