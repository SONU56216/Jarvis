# JARVIS AI Assistant

JARVIS is an advanced AI voice assistant created using Python.  
It is designed to work as a smart desktop assistant with voice interaction, automation, and AI-powered features.

## Features
- Voice commands
- AI conversation system
- Music player
- Notes and task manager
- Alarm & reminder system
- Mood tracking
- Phone control using ADB
- App launcher
- Code assistant
- Productivity tools

## Built With
- Python
- Tkinter
- SpeechRecognition
- pyttsx3
- Google Gemini AI
- OpenCV
- Pygame
